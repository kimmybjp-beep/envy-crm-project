Apple ENVY Reward CRM

คู่มือการใช้งานระบบ ENVY Rewards

สำหรับร้านค้า, Back Office Admin, การอนุมัติร้าน, การสแกน QR, การสะสมแต้ม, การแลกรางวัล, Dashboard และ Export ข้อมูล

ภาพรวมระบบ ร้านค้าใช้งาน Admin ใช้งาน QR Generator Dashboard / Data แก้ปัญหาเบื้องต้น

1. ภาพรวมระบบ

ร้านค้า สมัครร้าน, รออนุมัติ, Login ด้วยเบอร์โทร, สแกน QR เพื่อสะสมแต้ม, แลกของรางวัล
Back Office Admin อนุมัติร้าน, สร้าง QR batch, สร้าง reward, ดู dashboard, export data
ระบบกันสแกนซ้ำ QR code เดียวกันจะสแกนซ้ำใน tier เดียวกันไม่ได้ เพื่อกัน double count

จำง่ายๆ: ร้านค้าต้องได้รับสถานะ APPROVED ก่อน ถึงจะ Login และ Scan ได้

2. คู่มือฝั่งร้านค้า

2.1 สมัครร้านใหม่

  1. เปิดหน้าแรกของระบบ
  2. กดปุ่ม สมัครร้านใหม่
  3. กรอกชื่อร้าน, ชื่อเจ้าของร้าน, เบอร์โทร, เลือก Tier
  4. ใส่ URL รูปหน้าร้านถ้ามี
  5. กดส่งสมัครร้าน
  6. รอ Back Office อนุมัติ

2.2 Login เข้าระบบร้าน

  1. กด เข้าสู่ระบบร้านค้า
  2. กรอกเบอร์โทรที่ใช้สมัครร้าน
  3. ถ้าร้านอนุมัติแล้ว ระบบจะพาไปหน้า Home

2.3 หน้า Home ร้านค้า

  • แสดงชื่อร้าน
  • แสดงแต้มสะสม
  • ปุ่ม Scan QR
  • ปุ่ม Rewards สำหรับแลกของรางวัล
  • ข้อความประกาศจาก Admin

2.4 วิธีสแกน QR จากรูปในมือถือ

  1. เข้าเมนู Scan
  2. กด Choose File
  3. เลือกรูป QR ที่เซฟไว้ในมือถือ จะเลือกหลายรูปพร้อมกันได้
  4. ระบบอ่าน QR และแสดงจำนวนที่พร้อมส่ง
  5. กดบันทึก Scan
  6. ถ้าสำเร็จ แต้มร้านจะเพิ่มอัตโนมัติ

ถ้าขึ้นข้อความ “โค้ดนี้ถูกสแกนใน Tier ของคุณไปแล้ว!” แปลว่า QR นี้เคยถูกใช้ใน tier นี้แล้ว

3. คู่มือฝั่ง Back Office Admin

3.1 Login Admin

  1. เข้า /admin/login
  2. กรอกรหัส Admin Password
  3. ระบบจะพาไปหน้า Store Approval

3.2 อนุมัติร้าน

  1. เข้า /admin
  2. ดูร้านที่รออนุมัติใน Pending queue
  3. กด Approve เพื่อเปิดให้ร้าน Login และ Scan ได้
  4. กด Reject ถ้าไม่อนุมัติ

3.3 ส่งข้อความประกาศให้ร้าน

  1. เข้า /admin
  2. พิมพ์ข้อความในช่อง Admin broadcast message
  3. กด Publish
  4. ร้านค้าจะเห็นข้อความที่หน้า Home

4. QR Generator

  1. เข้า /admin/qr-generator
  2. กรอกชื่อ Distributor
  3. กรอกจำนวน QR ที่ต้องการสร้าง
  4. กด Generate ZIP
  5. ระบบจะบันทึก batch ไปที่ qr_batches
  6. ระบบจะบันทึก QR code ทุกตัวไปที่ qr_codes
  7. Browser จะดาวน์โหลด ZIP รูป QR สำหรับพิมพ์แจก

QR code เป็นรหัสสุ่ม 18 หลักแบบ unique ไม่ซ้ำกัน

5. Dashboard, Rewards และ Export Data

หน้า ใช้ทำอะไร
/admin/dashboard ดูภาพรวม store, approved, pending, scans, points และแยก tier
/admin/dashboard/DISTRIBUTOR ดูข้อมูลเฉพาะ Distributor
/admin/dashboard/TIER2 ดูข้อมูลเฉพาะ Tier 2
/admin/dashboard/TIER3 ดูข้อมูลเฉพาะ Tier 3
/admin/rewards สร้างและแก้ไข reward catalog, ดูคำขอแลกรางวัล
/admin/database ดูจำนวนแถว, preview raw data และ Export CSV เปิดใน Excel ได้

6. แก้ปัญหาเบื้องต้น

อาการ วิธีเช็ก
ร้าน Login ไม่ได้ เช็กว่าเบอร์โทรตรงกับที่สมัคร และ status เป็น APPROVED
สแกนแล้วขึ้นซ้ำ QR code นั้นเคยถูกสแกนใน tier เดียวกันแล้ว
QR Generator ไม่สำเร็จ เช็ก Supabase env ใน Vercel และรัน SQL fix สำหรับ QR table แล้วหรือยัง
Export ไม่ออก เช็กว่าเปิดจากหน้า Admin ที่ login แล้ว และตารางมีข้อมูล
